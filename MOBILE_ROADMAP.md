# Neon Defense Mobile-First Roadmap

## Goal
Make Neon Defense feel like a real phone game on a standard mobile screen, while keeping desktop support.

## Phase 1 — Phone Playability
- [ ] Make all in-game overlays phone-safe
- [ ] Rework tower info / upgrade popup positioning so it never covers the selected tower
- [ ] Build a true mobile HUD mode
- [ ] Keep Start Wave / Sell / Bestiary always reachable
- [ ] Increase touch forgiveness for placement and selection

## Phase 2 — Mobile UX
- [ ] Mobile-first tower interaction flow
- [ ] Thumb-friendly control spacing
- [ ] Better mobile rally placement for sentinels
- [ ] Compact upgrade actions and clearer selected state

## Phase 3 — Performance
- [ ] Reduce lag under heavy enemy + sentinel load
- [ ] Scale particles/effects under pressure
- [ ] Reduce expensive targeting loops
- [ ] Add quality scaling for slower devices

## Phase 4 — Mobile Retention
- [ ] Expand perk system
- [ ] Stronger boss progression
- [ ] Better run hooks / replayability
- [ ] App-friendly UI polish

## Immediate priority order
1. Tower popup / overlay safety
2. True mobile HUD mode
3. Mobile interaction flow
4. Performance pass
5. Progression expansion
