# Neon Defense

A fast-paced neon tower defense game with geometric art style.

## How to Play

1. Open `index.html` in any modern browser
2. Place towers to defend against waves of enemies
3. Earn gold from kills and complete waves
4. Choose perks after each level to power up
5. Survive as long as possible!

## Tower Types

- **Blaster** (💰50) - Fast firing, balanced damage
- **Sniper** (💰100) - Slow but high damage, long range
- **AOE** (💰150) - Area damage, slows enemies
- **Boat** (💰75) - Water-only, good range
- **Sentinel** (💰75) - Deploys mini-drones that block enemies

## Enemy Types

- **Basic** - Standard enemies
- **Fast** - Quick but fragile
- **Tank** - Slow, high HP, resistant to kinetic
- **Shield** - Energy shield, weak to pierce
- **Swarm** - Many weak, burn easily
- **Healer** - Heals nearby enemies
- **Stealth** - Hard to hit
- **Bosses** - Titan, Brood King, War Titan, Void Boss

## Controls

- **Click** to place tower
- **Click existing tower** to upgrade or sell
- **Start Wave** button to begin each wave
